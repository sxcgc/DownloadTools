# DownloadTools
想满带宽下载MODIS数据吗？试试这个肯定不会错！

# 需要的环境（工具）
1.IDM下载器（有可能请支持正版）

2.python 3.6（Anaconda 你值得拥有）

3.MODIS官网请求数据CSV文件（https://ladsweb.modaps.eosdis.nasa.gov）

# 需要的技能
1.了解简单的CMD 指令

2.会获取MODIS官网数据的CSV，编辑文件并保存

# 步骤
1.打开源文件，修改变量idmPath;modis_csv;outputPath,CSV的第一行见example文件。

2.运行程序pyidmTool.py
